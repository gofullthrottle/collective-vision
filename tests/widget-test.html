Instructions

Before testing:

  1. Start the dev server: npm run dev
  2. Seed the database (optional): npm run db:seed
  3. Click "Load Widget" above

Testing checklist:

  • Submit new feedback
  • Vote on existing items
  • Check mobile responsiveness (resize window)
  • Toggle dark mode to test contrast
  • Check browser console for errors

Embed code for production:

<script
  src="https://your-domain.com/widget.js"
  data-workspace="your-workspace"
  data-board="your-board"
></script>
